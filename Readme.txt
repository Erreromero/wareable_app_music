# Nombre del Proyecto

## Descripción
Este proyecto es una aplicación móvil que permite a los usuarios escuchar y gestionar canciones. La aplicación incluye funciones de autenticación, un reproductor de música, la capacidad de subir y gestionar canciones en la nube, y la opción de elegir entre modo claro y oscuro.

## Características
- **Modo Claro-Oscuro:** Los usuarios pueden elegir entre modo claro y oscuro.
- **Autenticación:** Inicio de sesión y registro utilizando Firebase.
- **Gestión de Canciones:** Subir, reproducir y gestionar canciones.
- **Favoritos:** Añadir y gestionar canciones favoritas.
- **Perfil del Usuario:** Visualizar y editar información del perfil del usuario.

## Instalación
Sigue estos pasos para configurar el proyecto en tu máquina local:

1. Clona el repositorio:
    ```bash
    git clone https://github.com/tu_usuario/tu_proyecto.git
    ```

2. Navega a la carpeta del proyecto:
    ```bash
    cd tu_proyecto
    ```

3. Instala las dependencias:
    ```bash
    flutter pub get
    ```

4. Configura Firebase para la autenticación y almacenamiento de datos.

## Uso
Para ejecutar la aplicación en tu dispositivo local:

1. Conecta tu dispositivo o inicia un emulador.
2. Ejecuta el proyecto:
    ```bash
    flutter run
    ```
